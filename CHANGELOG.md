# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-26

### Added

- Initial release of LinkSnip
- URL shortening with auto-generated codes
- Custom short codes support
- Analytics dashboard with authentication
- Click tracking with anonymized IPs
- Rate limiting to prevent abuse
- Docker support for easy deployment
- Comprehensive API documentation
- Health check endpoint
- MongoDB integration
- Responsive frontend UI
- Admin authentication for analytics
- Top referrers tracking
- Browser and OS analytics
- Daily click statistics
- Popular URLs ranking
- System-wide analytics
- URL management (create, read, delete)
- Automatic data retention (90 days for click events)
- Postman collection for API testing

### Security

- IP address anonymization
- HTTP Basic Authentication for analytics
- Environment variable configuration
- Rate limiting on URL creation
- Input validation and sanitization

### Documentation

- Complete README with setup instructions
- API documentation with examples
- Docker deployment guide
- Local development guide
- Contributing guidelines
- Security policy
- Code of conduct
- Deployment checklist
